# Terminal Tavern Website

Website for [Terminal Tavern](https://www.terminaltavern.com) and [PhotonBBS](https://github.com/fewtarius/photonbbs).

## Local Development

Open `index.html` in a browser. No build process, no dependencies.

## Deployment

Hosted on GitHub Pages. Push to `main` to deploy.
